
 1 уровень сложности: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
1. Переходим в терминал.
2. Проверяем настройки git при помощи команды git config --global -l
3.Если у вас при проверке нету user name и user email 
4. Тогда нужна заполнить эти поля git config --globol user.name и нужно указать свое имя. И также добавляем емайл, git config --globol user.email и записать свой емейл.
5. После того нужно зайти на GitHub тут нужно проверить есть ли ключ SSH, если у вас его нету его можно генерировать или вставить туда уже сушествуеший. 
6. В домашней директории набираем команду ls -al находим попку .ssh переходим в эту директорию и находим файл и смотрим что в этом файле cat id_rsa.pub. Скопировать то что находиться в файле и вставить этот ключ в Github.
7. После того вбираем протокол SSH и копируем силку в гит хабе.
8. Переходим в нужною директорию вводим команду git clone и вставляем скопированную силку.

Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
1.Берем выбраную директорию. 
2. git init
3. git status
4. git add .
5. git commit -m "commentary commit"
6. Создайом папку на гитхабе.
7. git remote add origin ссилка на репозиторий
8. git branch -M main(если старий гид)
9. Git push -u origin main
